Used APIs

Facebook SDK for JavaScript

    JavaScript SDK is an API that enables people to sign into a web application with their Facebook credentials. To implement Facebook login we will follow the steps:

    • Enter Our Redirect URL to take people to your successful-login page.
    • Check the login status to see if someone's already logged into your app. During this step, we will also check to see if someone has previously logged into our app, but is not currently logged in.
    • Log people in, either with the login button or with the login dialog, and ask for a set of data permissions.
    • Log people out to allow them to exit from our app.

JS Store

    JsStore is an API used to execute Database operation in browsers with JsStore. JsStore is an IndexedDB Wrapper. It makes IndexedDB super easy with its SQL like apis. We will use it to store data from users.

Google Data Studio

    Data Studio is an API used to represent a wide variely of data. The Data Studio Explorer streamlines the editing, viewing and data visualization experience. We can apply filters quickly, export our explorations to new or existing reports. We can even access the Explorer from any report to start exploring the data without modifying your report. We will use this API for statistics about alerts and allergies.

Diffbot

    Diffbot is a crowlbot that extract data from entire sites. Features:

    • Smart Processing : Automatically find and extract all article, product or other supported pages.
    • Structured Search : All crawls are instantly searchable using Search API, allowing to slice and dice the data by searching the Diffbot-structured fields. Sort by article date, filter by product price, search across your custom fields: it's all in there.
    • Wolf-Spider Fast : Distributed, world-class crawling infrastructure processing millions of pages daily.
    • Diversified IP Options : We can utilize the reserved fleet of proxy IPs, optionally upgrade to gain access to tens of thousands of unique IPs for truly diversified crawling or region/country-specific extraction.

Micro Data

  • Microdata HTML scheme

    Schema.org is a collaborative, community activity with a mission to create, maintain, and promote schemas for structured data on the Internet, on web pages, in email messages, and beyond.

    Schema.org vocabulary can be used with many different encodings, including RDFa, Microdata and JSON-LD. These vocabularies cover entities, relationships between entities and actions, and can easily be extended through a well-documented extension model.

    Most webmasters are familiar with HTML tags on their pages. Usually, HTML tags tell the browser how to display the information included in the tag.

    By adding additional tags to the HTML of your web pages—tags that say, "Hey search engine, this information describes this specific movie, or place, or person, or video"—you can help search engines and other applications better understand your content and display it in a useful, relevant way. Microdata is a set of tags, introduced with HTML5, that allows you to do this.
