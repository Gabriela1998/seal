Implementare

Facebook Login

Pentru logarea cu o retea de socializare am ales API-ul Facebook SDK pentru JavaScript. Deoarece aplicatia este in stadiul de development, platforma Facebook for developers ne ofera posibilitatea de a creea utilizatori de testare, generati random.

Aceasta functie se conecteaza cu platforma Facebook for developers prin intermediul id-ului aplicatiei, pentru a accesa de exemplu datele de test create sau informatii despre utilizatori.

In momentul in care utilizatorul acceseaza butonul de login, se apeleaza functia Check Login State: Astfel se apeleaza functia Status Change Callback, care verifica daca exista un utilizator logat cu Facebook pentru a se loga si in aplicatie si care apeleaza la randul ei functia testAPI.
